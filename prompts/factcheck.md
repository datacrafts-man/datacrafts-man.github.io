# Fact Check Prompt

아래 블로그 초안의 사실 관계를 검증하세요.

## 검증 대상

- 파일: {{FILE_PATH}}
- 카테고리: {{CATEGORY}}

## 검증 항목

1. **수치·통계** — 출처가 있는가? 최신 데이터인가?
2. **API/라이브러리** — 버전, 함수명, deprecated 여부
3. **코드 예제** — 문법 오류, 실행 가능 여부
4. **비교·주장** — "최고", "유일" 등 근거 없는 표현
5. **링크** — 깨진 URL, 잘못된 참조

## 출력 형식

```markdown
## Fact Check Report

### 통과
- ...

### 수정 필요
- [심각도: high/medium/low] 위치: ... / 문제: ... / 제안: ...

### 확인 불가 (작성자 확인 필요)
- ...
```

## 규칙

- 확실하지 않으면 "확인 필요"로 표시하고 단정하지 않는다.
- 수정 제안은 구체적인 문장 대안을 포함한다.
