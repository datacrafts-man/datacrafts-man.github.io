datacrafts-man 블로그가 정상적으로 동작하는지 확인하기 위한 첫 글입니다.

## 로컬 미리보기

```bash
hugo server -D
```

브라우저에서 `http://localhost:1313`으로 접속하면 사이트를 확인할 수 있습니다.
